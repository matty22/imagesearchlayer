Image Search Abstraction Layer

This Image Search Abstraction Layer application will take a search parameter at the /search endpoint in the URL and will return search results based on that parameter. You can also provide an offset on the end of your search to paginate the results.

If you navigate to the /latest endpoint, the application will return the most recent searches performed on the /search endpoint.

Enter a URL like this into your browser URL bar to search for MEAN stack images:

https://matty22imagesearchlayer.herokuapp.com/api/search/MEAN%20stack?offset=10

Example output:

{ An object full of MEAN stack images }

Enter a URL like this into your browser URL bar to see your most recent searches:

https://matty22imagesearchlayer.herokuapp.com/api/latest

Example output:

{ An object full of recent image searches }
